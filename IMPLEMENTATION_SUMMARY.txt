PROJECT OVERVIEW
----------------
Complete Python implementation of a system for detecting and diagnosing partial 
discharges (DP) in UHF signals (300 MHz - 1.5 GHz) with comprehensive signal 
processing, feature extraction, adaptive filtering, and validation capabilities.

CORE MODULES (8 files, ~2,200 lines)
------------------------------------
1. signal_processing.py (238 lines)
   - Bandpass and lowpass Butterworth filters
   - Normalization (z-score, minmax, RMS)
   - Hilbert transform envelope detection
   - Savitzky-Golay and median noise reduction

2. feature_extraction.py (238 lines)
   - Energy, RMS, residual calculations
   - Kurtosis and skewness (statistical features)
   - Frequency band analysis (3 UHF bands)
   - Spectral stability via STFT

3. traffic_light_classifier.py (286 lines)
   - Weighted feature combination
   - Dynamic threshold adjustment (percentiles)
   - 4-level classification (green/yellow/orange/red)
   - Diagnostic message generation

4. adaptive_filters.py (262 lines)
   - EWMA (Exponentially Weighted Moving Average)
   - Moving Average
   - Kalman Filter
   - LMS (Least Mean Squares)
   - RLS (Recursive Least Squares)

5. validation.py (307 lines)
   - SNR, MSE, RMSE calculations
   - Event detection and matching
   - TP/FP/FN metrics
   - Precision, Recall, F1-Score
   - Comparative analysis tables

6. dp_detection_system.py (358 lines)
   - Main integrated system
   - Complete processing pipeline
   - Diagnostic report generation
   - Synthetic signal generator
   - Historical data management
